# Laravel ACL implementation

1- Edit the ".env" file with your "DB_DATABASE", "DB_USERNAME" and "DB_PASSWORD" .

2- run the folowing commands:

	$ composer install
	$ php artisan migrate
	$ php artisan db:seed
	$ php artisan serve
